## velda config set

Save a config item

```
velda config set --option=value [flags]
```

### Options

```
      --broker string               The address of broker
      --default-instance string     Configure default instance
  -h, --help                        help for set
      --identity-file string        The identity file to use for authentication
      --jump-identity-file string   The identity file to use for jump server authentication
      --jump-proxy string           SSH jump server in user@host format
```

### Options inherited from parent commands

```
      --config_dir string   config directory. Defaults to env VELDA_CONFIG_DIR or ~/.config/velda
      --debug               Enable debug mode
      --profile string      The user profile to use.
```
