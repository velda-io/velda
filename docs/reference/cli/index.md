# Velda CLI Reference

The `velda` command-line interface provides full control over your Velda cluster — launching workloads, managing instances, configuring authentication, and more.

## Installation

The CLI binary is distributed as a single static executable. Download the latest release from [GitHub Releases](https://github.com/velda-io/velda/releases) and place it on your `PATH`.

```bash
# Linux / macOS (amd64)
curl -L https://github.com/velda-io/velda/releases/latest/download/velda-linux-amd64 -o velda
chmod +x velda
sudo mv velda /usr/local/bin/
```
